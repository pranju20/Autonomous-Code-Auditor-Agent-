# Autonomous-Code-Auditor-Agent-

Developed a stateful agentic system using LangGraph to automate code reviews and autonomous bug-fixing by identifying vulnerabilities in real-time.Implemented a multi-node architecture featuring self-correction loops that iteratively audit, repair, and re-validate code to ensure high reliability and logic accuracy.I built an agentic AI system that performs automated code auditing using a multi-agent architecture. It uses LangGraph for workflow orchestration, Pydantic for structured outputs, and Gemini LLM for reasoning. The system includes retry mechanisms, guardrails, and validation to ensure reliability, making it close to production-grade
